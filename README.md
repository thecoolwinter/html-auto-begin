# html-auto-begin
Kickstart your html document with a simple key command.

Just press control-shift-h and some html begining text will pop up.

html text:
```
<!DOCTYPE html>
<html lang="en">
<head>

  <title>New Html Website</title>

</head>
<body>

</body>
</html>
```
